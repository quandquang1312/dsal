## data structure
### prefix, suffix
prefix sum of a ($a_0, a_1, a_2, ..., a_n$) is an array represent the cumulative sum of an array. With prefix sum at $i$ is the cumulative sum from $a_0$ to $a_i$.
$$prefixSum[j] = \sum_{i=0}^{j}a[i]$$
